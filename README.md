📌 Overview

This project focused on implementing Role-Based Access Control (RBAC) using Windows Server to understand how organizations manage user permissions, enforce least privilege access, and maintain secure identity management systems.

The project involved configuring a Domain Controller, creating Organizational Units (OU), managing users and roles, assigning permissions based on responsibilities, and validating access control policies. This hands-on experience provided practical exposure to Active Directory administration, access management, and enterprise security practices.
